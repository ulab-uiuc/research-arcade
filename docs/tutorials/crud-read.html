Best Practices

  • Use indexed properties for filtering to improve query performance
  • Limit result sets using pagination for large queries
  • Be specific with relationship types when traversing to reduce search space
  • Cache frequently accessed nodes and relationships
  • Use projection to retrieve only needed properties
  • Consider query complexity and optimize for common access patterns
